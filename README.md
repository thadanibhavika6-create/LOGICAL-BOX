# LOGICAL-BOX

This project is a simple console-based application that demonstrates basic programming concepts like loops, conditionals, and user input/output.

🔹 Features
Pattern Generator:

User enters number of rows.

Program prints a triangle pattern of * increasing line by line.

Number Analyzer:

User enters a start and end range.

Program checks each number → odd or even.

Calculates and displays the sum of all numbers in the range.

Exit Option:

Allows user to quit the program safely.
